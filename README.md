# 📈 Sales Forecasting & Demand Intelligence Dashboard

An end-to-end sales forecasting and business intelligence project developed during my internship.

## Project Overview

This project analyzes historical sales data to generate business insights, forecast future sales, detect anomalies, and segment products based on demand.

The dashboard was built using Streamlit and integrates multiple forecasting techniques to support business decision-making.

---

## Features

- 📊 Interactive Sales Dashboard
- 🔮 Sales Forecasting
- 🤖 XGBoost Forecasting
- 📈 SARIMA Forecasting
- 📉 Prophet Forecasting
- 🚨 Anomaly Detection
- 📦 Product Demand Segmentation
- 📋 Interactive Filters
- 📥 Download Forecast Results
